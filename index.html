Project 01 · React + AI · Prototype Tested Locally
AI Strategy Engine
Built for Bluetik Marketing Agency · 2026

A single-page React application that replaces hours of manual strategy work with a 2-minute AI pipeline. The analyst pastes a client's Instagram URL — the app uses Claude AI to find the top 3 real competitors, scrapes all four profiles via Apify, and then feeds everything into Claude to generate a complete, client-specific campaign strategy and content kit.

Supports PDF intake form uploads — Claude reads and auto-fills the entire client brief. Covers 13 industries across India, operates in two modes (URL + Manual), and outputs Reels scripts, carousel copy, ad copy, captions, and WhatsApp follow-up messages — all tailored to the specific client, not generic templates.

Architecture: Fully client-side React app · No backend server · Vite proxy rewrites /api/* → Anthropic · Apify called directly · PDF.js loaded dynamically from CDN
Scale: ~₹4–11 per full client analysis · 150–200 analyses on $10 credit · Used by agency strategists, account managers & sales team
Codebase: ~1,560-line App.jsx · 4 Claude API calls per session · JSON repair logic · Fallback strategy if API fails

Project 02 · Node.js + WhatsApp + AI · Prototype Tested Locally
AI Saloon Management System
Client: Kerala Saloon Chain — 15 Branches · 2026

A full WhatsApp-native AI booking and management platform for a 15-branch Kerala saloon chain serving 250 customers/day. Customers message any branch's WhatsApp number in Malayalam or English — Claude AI handles the full conversation: booking appointments, answering pricing queries, sending 24-hour reminders, and collecting post-service feedback.

Built the complete production architecture from scratch: 5-layer system (WATI → Node.js → Claude AI → PostgreSQL → React dashboards), full 10-phase build roadmap, INR infrastructure pricing for client pitch, risk matrix, 6-month post-launch plan, and ROI case — break-even in under 10 months.

Architecture: WATI (15 WhatsApp numbers) → Node.js + Express (Railway) → Claude Haiku/Sonnet → PostgreSQL (Supabase) → React dashboards (Vercel) — no Redis needed at this scale
AI Design: Custom per-branch system prompts · Intent classifier returning structured JSON · Last-5-message context memory · 200+ Malayalam test messages · Escalation to manager on 2 consecutive unknown intents
Business case: ₹6,00,000 one-time build · ₹27,000/month recurring · ₹3.60 cost per customer · Staff time saving = ₹56,250/month · ROI positive from Month 2
